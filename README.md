# Electron

### Descripción

Controlador de URL's en una aplicación local creada con Electron basado en código node.js

----

### Instalación

    npm i

----

### Comandos

Ejecutar el proyecto

    electron .

Crear la aplicación local

    electron-builder

Actualización de versión desde GitHub releases para windows

    electron-builder -w
